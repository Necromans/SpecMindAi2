# 🚀 SpecMind

**SpecMind** — это интеллектуальная система для анализа и улучшения технических заданий (НТЗ) научных проектов с использованием искусственного интеллекта.

> Мы не просто анализируем документы — мы помогаем их улучшать.

---

## 🔥 Основные возможности

- 📄 Анализ документов (PDF, DOCX, TXT)
- 🧠 AI-оценка качества технического задания
- 📊 Выявление проблем и рекомендаций
- 💬 Чат с AI для доработки документа
- 📥 Формирование отчётов

---

## ⚙️ Быстрый запуск (локально)

### 1. Установить .NET SDK

Скачать:
https://dotnet.microsoft.com/download

Проверить установку:

dotnet --version
Открыть терминал (CMD / PowerShell) и выполнить:

git clone https://github.com/Necromans/SpecMindAi.git
cd SpecMindAi

Восстановить зависимости
dotnet restore

Запустить приложение
dotnet run --urls "http://0.0.0.0:5000"

Открыть в браузере

После запуска в консоли появится сообщение:

Now listening on: http://0.0.0.0:5000

Перейдите по ссылке:

👉 http://localhost:5000
